# AIお守り

悩みを託すと、AIが癒やしの言葉と共にふわふわとした世界観を持つ3Dお守りを生成する癒やしのWebアプリです。

✧ 作品コンセプト
現代のストレス社会に寄り添う「デジタルなお守り」を提案します。
パステルカラーの色彩と浮遊感のあるパーティクルで癒やしの空間を演出しました。

✧ 主な機能
- AIメッセージ生成: LLM APIを活用し、ユーザーの悩みに寄り添う独自のメッセージを生成。
- 3Dお守り生成: p5.jsを使用し、AIが選んだ色と形状で世界に一つのお守りを作成。
- スマホ連携インタラクション: Socket.ioとスマホの傾きセンサーを同期させ、PC画面上のお守りを手元で操るような体験を実現。

✧ 使い方
1. PCの入力欄に悩みや願いを入力します。
2. お守りが生成されたら、サイドバーのQRコードをスマホで読み取ります。
3. スマホ画面で「接続」を押し、スマホを傾けるとお守りが揺れ動きます。

✧ 使用技術
- Frontend: p5.js, HTML/CSS, JavaScript
- Backend: Node.js, Express, Socket.io
- AI: OpenAI GPT-4o-mini
- Deployment: Render
